Project 1 - Entrepreneurship Project

This was the end-of-year project for CSE, and was by far the most unique, intense, and educational of all that I did in this class this year. Spanning nearly a month from the start of spring break to the start of May, it required me and my groupmates to independently create a product that addressed a problem in society in a meaningful way. My role as front-end developer meant that I had to develop the website and UI, but this was far from easy. The main challenge was finding a way to integrate the backend code into the frontend, and through experimentation and guess-and-check, we were able to find a way to do so that was both simple and elegant.
